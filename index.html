Flexbox y Grid

Flexbox y Grid son dos técnicas de diseño web importantes que han revolucionado la forma en que creamos diseños web flexibles y responsivos.

Flexbox se introdujo en 2009 y Grid en 2011 como propuestas de diseño CSS3. Ambas tecnologías han sido ampliamente adoptadas por la industria del desarrollo web debido a su capacidad para simplificar la creación de diseños complejos y mejorar la capacidad de respuesta de los sitios web.

Historia



Flexbox, o Flexible Box Layout, es un modelo de diseño en CSS que se utiliza para diseñar diseños más flexibles y eficientes en términos de alineación y distribución de elementos en un contenedor. Aquí tienes una descripción de los conceptos clave en Flexbox:

Contenedor Flex (Flex Container)

El contenedor flex es el elemento principal que contiene un conjunto de elementos flexibles

↻

Que se organizan y distribuyen dentro de él. Se establece como un elemento padre utilizando la propiedad display: flex; o display: inline-flex;.

Elementos Flexibles (Flex Items)

Los elementos flexibles son los elementos hijos directos del contenedor flex que se distribuyen y

↻

Ajustan dentro del contenedor. Cada elemento flexible puede tener su propio ancho, alto y margen, pero se ajusta dentro del contenedor flex según las reglas de Flexbox.

Eje Principal
(Main Axis)

El eje principal es el principal eje de distribución en un contenedor flex.

↻

Cambias la dirección del eje principal estableciendo la propiedad flex-direction. Ejemplo, flex-direction: column; colocaría el eje principal de forma vertical, de arriba a abajo.

Eje Cruzado
(Cross Axis)

El eje cruzado es el eje perpendicular al eje principal dentro de un contenedor flex.

↻

Se utiliza para controlar la alineación vertical de los elementos flexibles dentro del contenedor.


En resumen, Flexbox es un modelo de diseño CSS que se centra en la distribución y alineación de elementos flexibles dentro de un contenedor flex. El contenedor flex define cómo se disponen los elementos flexibles en el eje principal y cruzado, lo que permite crear diseños flexibles y responsivos de manera eficiente. Los conceptos de eje principal y cruzado son especialmente importantes para comprender cómo se alinean y distribuyen los elementos en un diseño Flexbox.

Conceptos Fundamentales de Grid


Grid es un sistema de diseño bidimensional que permite crear diseños de cuadrícula complejos y alineaciones precisas de elementos en una página web.

Contenedor de Cuadrícula

El contenedor de cuadrícula es un elemento HTML que se convierte en un contenedor de cuadrícula utilizando la

↻

propiedad CSS `display: grid`. Dentro de este contenedor, puedes colocar elementos que se organizarán en una cuadrícula.

Filas y Columnas

En una cuadrícula, puedes definir el número de filas y columnas utilizando las propiedades `grid-template-rows` y `grid-template-columns`. una.

↻

Por ejemplo, `grid-template-rows: repeat(2, 100px)` crea dos filas de 100px de altura cada

Template Areas

La propiedad `grid-template-areas` te permite definir un esquema visual de tu cuadrícula utilizando nombres de áreas.

↻

Esto facilita la organización de elementos en la cuadrícula según el diseño deseado.

Posicionamiento de Elementos

Los elementos dentro de la cuadrícula se posicionan utilizando propiedades como `grid-row` y `grid-column`. ↻

Puedes especificar en qué fila y columna debe aparecer cada elemento

Ventajas de Grid: Facilita la creación de diseños complejos y responsivos. Ofrece un mayor control sobre la disposición de elementos en comparación con otros sistemas de diseño CSS. Simplifica la alineación y distribución de contenido en filas y columnas.

¿Que es Flexbox y Grid?

Flexbox y Grid son dos modelos de diseño en CSS que permiten crear diseños de páginas web complejos y flexibles. Aunque ambos tienen como objetivo facilitar la maquetación de elementos en una página web, tienen diferencias clave en términos de su estructura y cómo se utilizan.

  • 1. Estructura:
    • Flexbox: Se centra en la distribución de elementos en una sola dirección, ya sea en filas (horizontal) o columnas (vertical). Es ideal para diseñar diseños flexibles y escalables en una sola dimensión.
    • Grid: Permite crear diseños bidimensionales con filas y columnas, permitiendo un control más preciso sobre la posición y el tamaño de los elementos en la página
  • 2. Dirección de diseño:
    • Flexbox: Ideal para diseños unidimensionales, donde los elementos se alinean en una fila o columna y se ajustan según sea necesario.
    • Grid: Excelente para diseños bidimensionales, donde se pueden definir filas y columnas específicas para colocar elementos en una cuadrícula.
  • 3. Control sobre los elementos:
    • Flexbox: Proporciona un control más simple y rápido sobre la alineación, distribución y reordenamiento de elementos en una fila o columna.
    • Grid: Ofrece un control preciso y completo sobre la posición y el tamaño de los elementos en ambas direcciones (filas y columnas).
  • 4. Escenarios de uso:
    • Flexbox: Es ideal para elementos secundarios de un contenedor que necesitan ser distribuidos y alineados en una sola dirección, como barras de navegación, listas de elementos o formularios.
    • Grid: Se utiliza cuando se necesita un diseño más complejo y estructurado para todo el diseño de la página, como la maquetación de páginas completas, secciones complejas y módulos de diseño.

Diferencias clave entre Flexbox y Grid:

Usos Flexbox

.nav-container { display: flex; justify-content: space-around; }

.list-container { display: flex; flex-direction: column; }

.form-container { display: flex; flex-direction: column; }

.gallery-container { display: flex; flex-wrap: wrap; }

.content-container { display: flex; justify-content: space-between; }

Usos Grid

.page-container { display: grid; grid-template-columns: 1fr 2fr 1fr; grid-template-rows: 100px 200px; grid-gap: 10px; }

.section-container { display: grid; grid-template-columns: repeat(3, 1fr); grid-gap: 20px; }

.product-grid { display: grid; grid-template-columns: repeat(auto-fill, minmax(150px, 1fr)); grid-gap: 10px; }

.table-container { display: grid; grid-template-columns: repeat(3, 1fr); grid-gap: 10px; }

.page-layout { display: grid; grid-template-rows: auto 1fr auto; }

En resumen, Flexbox es excelente para diseños unidimensionales, como alinear elementos en una fila o columna, mientras que Grid es más adecuado para diseños bidimensionales, permitiendo crear estructuras complejas de filas y columnas. La elección entre Flexbox y Grid dependerá del diseño que necesites y si es un diseño más simple y lineal (Flexbox) o más complejo y estructurado (Grid).

-Andres Rojas A.

Consejos y trucos

Flexbox


Centrado fácil

Centrar elementos vertical y horizontalmente es muy fácil con Flexbox: justify-content: center y align-items: center.

flex-wrap para Multilínea

Si quieres que los ítems pasen a la siguiente línea en vez de encogerse, usa flex-wrap: wrap.

Ordenar ítems

La propiedad order es muy útil para cambiar el orden visual de los ítems sin cambiar el HTML.

No Olvides el Eje Cruzado

Es fácil enfocarse solo en el eje principal (horizontal en row, vertical en column), pero las propiedades que afectan el eje cruzado (como align-items) son igual de valiosas.

Uso de flex: shorthand

En lugar de definir flex-grow, flex-shrink, y flex-basis por separado, utiliza la notación resumida para ahorrar tiempo y espacio.

Uso Estratégico de flex-grow, flex-shrink, y flex-basis

Entender estas propiedades a fondo te permite controlar cómo tus ítems se adaptan a diferentes tamaños de contenedor. Por ejemplo, usar flex-grow: 1 en todos los ítems les permite crecer uniformemente para llenar el espacio.

CSS Grid


Diseño con grid-template-areas

Para diseños más complejos, define áreas en tu grid. Esto te permite visualizar fácilmente la estructura de tu diseño y mover áreas completas redefiniendo grid-template-areas.

Espaciado con gap

En lugar de utilizar márgenes, utiliza gap (anteriormente grid-gap) para espaciar uniformemente tus ítems en el grid.

Función repeat()

Usa repeat() para definir múltiples columnas o filas del mismo tamaño. Ejemplo:
grid-template-columns: repeat(3, 1fr).

Utiliza fr para distribución uniforme

La unidad fr es excelente para crear diseños que distribuyan espacio uniformemente entre ítems.

Grids anidados

Anida un grid dentro de otro para lograr diseños más complejos. Por ejemplo, un ítem de grid puede convertirse en un contenedor de grid por sí mismo.

Uso de Líneas Nombradas

Nombrar líneas en tu grid puede hacer que tu CSS sea más legible y te permite reposicionar ítems con facilidad.

Función minmax() para responsividad

Esta función es esencial para crear diseños adaptativos, permitiendo que las columnas y filas tengan dimensiones mínimas y máximas.

Ejercicios y desafíos

¡Utiliza estos desafíos para mejorar con Flexbox y Grid!

Flexbox

Nivel: Principiante
Crea un contenedor con 3 cajas dentro que se distribuyan horizontalmente y tengan el mismo espacio entre ellas.

justify-content: space-between;

Nivel: Intermedio
Crea una barra de navegación con 5 links. El primer y último link deben estar en los extremos opuestos de la barra.

justify-content: space-between;

Nivel: Intermedio
Crea una galería de imágenes donde las imágenes se distribuyan en 2 líneas y estén centradas.

flex-wrap: wrap;
justify-content: center;

Nivel: Avanzado
Crea un diseño de tarjeta con imagen, título, texto y botón. La imagen debe estar a la izquierda y el resto del contenido a la derecha, alineado verticalmente.

Utiliza flex-direction: row; y align-items: center; para el contenido.

Grid

Nivel: Principiante
Diseña una simple grilla de 3x3 cajas del mismo tamaño.

grid-template-columns: repeat(3, 1fr);
grid-template-rows: repeat(3, 1fr);

Nivel: Intermedio
Crea un layout de artículo con imagen a la izquierda, título y descripción a la derecha, y contenido debajo abarcando ambas columnas.

grid-template-areas: 'img title' 'img desc' 'content content';

Nivel: Intermedio
Diseña un encabezado de sitio web con un logo a la izquierda, navegación en el centro y un botón a la derecha.

grid-template-areas: 'logo nav btn';

Nivel: Avanzado
Diseña una grilla responsiva que muestra 4 columnas en pantallas grandes, 2 en medianas y 1 en pequeñas.

grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));
